# Project Title
LGU Dashboard System

## 🌐 Live Demo


## 📌 Description
This project is a responsive web application designed to provide a modern admin dashboard experience for local government unit (LGU) style workflows, including user authentication, file management, reporting, and admin user controls.

## ✨ Features
- User-friendly interface
- Responsive mobile + desktop design
- Dashboard overview cards and activity stream
- File/data management with search, add, edit, delete
- Report generation with filtering
- Admin user management (role assignment, delete)
- Dark mode toggle
- Toast notifications

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- AI Tools (for design assistance)
